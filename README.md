## HermesHub - Nemt

A project to manage trips for NEMT companies. Prioritized features:

- Automated member/passenger confirmation through SMS [ON HOLD]
- Automated trip confirmation through SMS [ON HOLD]
- Automated trip cancellation through SMS [ON HOLD]
- Automated trip rescheduling through SMS [ON HOLD]
- Export trips to CSV files
- Email input from insurances
- Email output to insurances
